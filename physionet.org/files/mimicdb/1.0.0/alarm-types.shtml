This is a list of alarm messages recorded in the .al files of the MIMIC Database. The messages were originally produced by the HP CMS (Merlin) bedside ICU monitors from which the MIMIC database records were obtained. Note that this is not a complete list of all possible alarm messages; it contains only those that were actually recorded in the MIMIC Database.

Many of the messages include numeric values. In this list, m represents a value measured and reported by the monitor, and n represents an alarm limit. These values are integers (except for STi and Tblood, where m.m and n.n indicate values with one digit to the right of a decimal point). Values may be negative. In a few cases (Pi and STi), i represents a channel number.
